# Disciplinary Committee

Этот проект является информационной системой документооборота, предназначенный для использования в Дисциплинарном комитете БСС СПбГУ. На данный момент он находится на раннем этапе разработки.

> В связи с утечкой в изначальный репозиторий чувствительной информации (сертификаты, пароли) он был оперативно удалён, а его содержимое без истории коммитов перенесено сюда.

> CI сложно реализовать на GitHub, так как для сборки используются .env-файлы, хранящие пароли от баз данных. Реализовано в копии репозитория, хранящейся на закрытом GitLab.

Различная документация, в т.ч. по вопросам лицензии размещена на Wiki.

Если Вы хотите внести вклад в развитие этого проекта, и Вы являетесь студентом СПбГУ, вступайте в Дисциплинарный комитет!💚
